Política de
Privacidade

Última atualização: maio de 2025  ·  Versão 1.0

1. Introdução

O MyWodScore ("nós", "nosso") é um aplicativo de timer e registro de PRs (personal records) para treinos. Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações dos usuários que utilizam nosso aplicativo.

Ao utilizar o MyWodScore, você concorda com as práticas descritas neste documento.

2. Informações que Coletamos

O MyWodScore coleta apenas as informações estritamente necessárias para o funcionamento do aplicativo:

  • ID do usuário — identificador único gerado pelo sistema para associar seus registros e configurações à sua conta.
  • ID do dispositivo — identificador do dispositivo utilizado para garantir o correto funcionamento do app e sincronização local.
Não coletamos nome, e-mail, localização, dados de saúde, contatos, histórico de navegação, dados financeiros ou qualquer outra informação pessoal além das listadas acima.

3. Finalidade do Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Garantir o funcionamento correto das funcionalidades do app (timer, registro de PRs)
  • Identificar a sessão do usuário para manter preferências e registros salvos
  • Assegurar a integridade e continuidade do uso entre sessões no mesmo dispositivo

Não utilizamos os dados para fins de publicidade, marketing ou compartilhamento com terceiros.

4. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos suas informações com terceiros, exceto quando exigido por lei ou necessário para o funcionamento técnico do aplicativo (ex.: serviços de infraestrutura da Apple).

5. Retenção de Dados

Os dados são armazenados localmente no dispositivo ou por período necessário para o funcionamento do app. Ao desinstalar o aplicativo, os dados locais são removidos automaticamente.

6. Segurança

Adotamos medidas técnicas razoáveis para proteger as informações dos usuários contra acesso não autorizado, alteração ou destruição. Como coletamos um volume mínimo de dados, o risco inerente é reduzido.

7. Privacidade de Menores

O MyWodScore não é direcionado a crianças menores de 13 anos. Não coletamos intencionalmente informações de menores. Caso identifique uso indevido, entre em contato conosco.

8. Seus Direitos

Você tem o direito de:

  • Solicitar informações sobre os dados que possuímos associados ao seu ID
  • Solicitar a exclusão dos seus dados
  • Desinstalar o aplicativo a qualquer momento, removendo os dados locais

Para exercer esses direitos, entre em contato pelo e-mail abaixo.

9. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos os usuários sobre mudanças significativas por meio de atualização da data no topo deste documento ou por aviso dentro do aplicativo.
